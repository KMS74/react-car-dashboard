## Technologies Used
- Vite(Next Generation Frontend Tooling)
- React
- React router dom
- Recharts(For data visualization)
- CSS
## Installation and Setup Instructions

#### Example:

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

Installation:

`npm install`

To Start Server:

`npm run dev`

To Visit App:

`http://127.0.0.1:5173/`
